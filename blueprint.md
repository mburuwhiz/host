# **App Name**: TWOEM Deploy

## Core Features:

- Interactive User Authentication & Onboarding: Provide a modern login/signup experience featuring email/password authentication (Argon2 hashing), email verification via a 'Verification Gate', and social logins (Google, GitHub, GitLab). The UI will incorporate interactive 'peek-a-boo' elements (responsive to typing, cursor movement) on desktop. User signup includes detailed profile collection (Full Name, Date of Birth, Company, Position, Phone, Primary Role, Main Language, Git Username) and supports an 'ADMIN_EMAIL' bootstrap for initial SuperAdmin setup.
- AI-Powered Application Detection & Deployment: Connect Git repositories and leverage an AI tool to intelligently analyze repository contents, providing auto-suggested build commands, start commands, and root directories. The system will then orchestrate atomic (Blue-Green) deployments to isolated Docker containers, ensuring zero-downtime application updates.
- Real-time Application Observability & Web Console: Deliver a comprehensive monitoring suite including live streaming application logs (stdout/stderr) and an interactive web terminal ('xterm.js') for direct container access. Visualize real-time CPU, RAM, network, and disk utilization metrics through interactive charts, providing critical insights into app performance.
- Advanced Environment Variable Management: Implement a 'Triple-Save' system for managing application environment variables. Changes can be saved to the database, applied through graceful container restarts (for non-build related config), or integrated into a full atomic rebuild process (for build-time dependent variables).
- Dynamic Networking & Automated SSL: Empower users to add and verify custom domains via clear dashboard instructions for DNS configuration. The platform will automatically provision and renew free SSL/TLS certificates from Let's Encrypt, guaranteeing secure HTTPS traffic for all deployed applications through a dynamic reverse proxy (Traefik/Nginx).
- Comprehensive Multi-Tenant Team Management: Enable users to create and manage multiple teams, invite members with defined roles (Owner, Developer, Viewer), and oversee team-specific settings, billing, resource quotas, and detailed audit logs to track all team activities.
- Integrated Support Ticketing System: Provide an in-dashboard support system where users can open and track support cases. The system will feature threaded conversations, email notifications for updates, and a dedicated admin view for managing and responding to all open tickets.

## Style Guidelines:

- The palette is designed for a light theme, based on a clean, modern aesthetic. The primary accent color is Emerald Green (#10b981), conveying growth and stability. This is complemented by Sky Blue (#0ea5e9) for secondary actions and interactive highlights, symbolizing clarity and innovation.
- The background color is a very light, desaturated green tint (#EDF8F4), providing a bright and airy base that harmonizes with the primary Emerald Green.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-savvy, and expressive feel. Body font: 'Inter' (sans-serif) for its neutral, objective readability across all content.
- Use elegant, minimalist, and clear iconography across the platform, ensuring icons are purposeful and consistent with the ultra-modern design aesthetic. Icons should visually complement the Emerald Green and Sky Blue palette.
- Desktop interfaces will utilize a persistent sidebar for primary navigation, maintaining quick access to main sections. Mobile interfaces will feature a responsive design with either bottom-tab navigation or a hamburger menu for optimal usability. An 8px grid system will ensure precise spacing and alignment, complemented by subtle shadows, gradients, and a uniform 16px border-radius for rounded corners, enhancing depth and premium feel.
- Incorporate purposeful and lightweight animations to enrich the user experience. This includes subtle scroll reveals for content, micro-interactions for button presses and form feedback, skeleton loaders for data-intensive sections (like metrics and logs) to indicate loading states gracefully, and smooth transitions between page navigations and deployment stages. The 'peek-a-boo' interactivity on login/signup forms will animate elements responsively to user input (typing, cursor movement, hover effects).
